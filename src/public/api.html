My Diabetes JS

API Documentation

v1.0

My Diabetes JS is a web platform to control your daily glucose level.

API developed in NodeJS with the ExpressJS framework managing the web application and KnexJS to build queries and datasource connection.

Security

METHOD RESOURCE DESCRIPTION REQUEST FIELDS ACCESS TOKEN
POST /security/register Register a new user. name, email, login, password, role_id
POST /security/login Login with credentials. login, password

Users

METHOD RESOURCE DESCRIPTION REQUEST FIELDS ACCESS TOKEN
GET /users Get all users.
GET /users/{id} Get user by id.
PUT /users/{id} Update a user based on id. name, email, login, password, role_id
DELETE /users/{id} Delete user by id.

Glucose

METHOD RESOURCE DESCRIPTION REQUEST FIELDS ACCESS TOKEN
GET /glucose Get all glucose readings.
GET /glucose/{id} Get a glucose reading by id.
GET /glucose/user/{id} Get glucose readings of a specific user.
GET /glucose/markermeal/{id} Get all glucose readings by markermeal id.
POST /glucose Add glucose reading. userId, glucose, unityId, date, hour, markerMealId
PUT /glucose Update glucose reading by id. glucose, unityId, date, hour, markerMealId
DELETE /glucose/user/id Delete glucose by user id.
DELETE /glucose/{id} Delete a glucose reading by id. glucose, unityId, date, hour, markerMealId

Marker meal

METHOD RESOURCE DESCRIPTION REQUEST FIELDS ACCESS TOKEN
GET /markermeal Get all marker meals.
GET /markermeal/{id} Get a marker meal by id.
DELETE /markermeal/{id} Delete a marker meal by id.
POST /markermeal/{id} Create a new marker meal. description
PUT /markermeal/{id} Update a markermeal description by id. description
